# lily58
For SplitKB Aurora Lily58. This repo builds both for independent usage and with a dongle. There are 4 artifacts:
 * `splitkb_aurora_lily58_left_dongled.uf2`: Dongletized Left
 * `splitkb_aurora_lily58_left-nice_nano_v2.uf2`: Independent Left
 * `splitkb_aurora_lily58_right-nice_nano_v2-zmk.uf2`: Both Independent & Dongletized Right
 * `settings_reset-nice_nano_v2-zmk.uf2`: Settings Reset (universal use)
